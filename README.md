## 🎓 Classroom Exercises & Fundamentals
ส่วนนี้รวบรวมโปรเจกต์และแบบฝึกหัดที่พัฒนาขึ้นในระหว่างการศึกษา ซึ่งเป็นรากฐานสำคัญในการฝึกฝนทักษะการแก้ปัญหาเชิงตรรกะ (Logical Thinking) และการออกแบบอัลกอริทึม (Algorithm Design) ด้วยภาษา Python

### 🛠 Skills Developed:
* **Logic & Control Flow**: ฝึกฝนการใช้คำสั่งเงื่อนไข (Conditionals) และการวนซ้ำ (Loops) ในสถานการณ์ที่หลากหลาย
* **Data Processing**: การจัดการข้อมูลตัวเลขและการคำนวณพื้นฐานทางคณิตศาสตร์
* **Report & Automation**: การเขียนโค้ดเพื่อสร้างรายงานสรุปผลข้อมูลแบบอัตโนมัติ
